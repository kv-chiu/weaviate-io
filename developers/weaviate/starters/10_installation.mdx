## Overview

Weaviate can be configured and deployed in many different ways. This page suggests some setups for different scenarios.

:::tip Use a Weaviate client library
In all cases, we recommend using Weaviate client libraries to interact with your Weaviate instance.
:::

## Suggested setups

### For evaluation: WCS / Embedded

You can get going in just minutes with:

- A free Weaviate Cloud Services (WCS) sandbox instance, or
- Embedded Weaviate.

:::note WCS sandboxes will expire in 14 days.
:::

And an inference-API based vectorizer (e.g. `text2vec-cohere`, `text2vec-huggingface` or `text2vec-openai`).

:::tip Start here:
- [Installation/WCS](../installation/weaviate-cloud-services.md)
- [Installation/Embedded](../installation/embedded.md)
:::

### For development: Docker / WCS

Using Weaviate open source with **Docker** will allow you to configure Weaviate the way you want, while still being easy to set up.

Alternatively, using a paid **WCS** instance may be another convenient option, although you should keep in mind that it will not be as flexible. For example, you will not be able to enable certain modules, such as to use a local vectorizer.

:::tip Start here:
- [Installation/Docker](../installation/docker-compose.md)
- [Installation/WCS](../installation/weaviate-cloud-services.md)
:::

### For production use: WCS / Kubernetes / Hybrid SaaS

Using **WCS** with a paid SLA tier is a convenient option for production use that will be easy to get going with. It also means you do not have to manage your infrastructure.

Alternatively, you can also use Weaviate open source and deploy it with **Kubernetes**. This will allow you to configure Weaviate the way you want, and leverage Kubernetes' devops capabilities.

Last but not least, we offer what we call **Hybrid SaaS** options for our customers. This means that we will deploy Weaviate on your infrastructure, and manage it for you. This is a good option if you want to use Weaviate in production, but do not want to manage it yourself.

:::tip Start here:
- [Installation/WCS](../installation/weaviate-cloud-services.md)
- [Installation/Kubernetes](../installation/kubernetes.md)
- [Contact us about Hybrid SaaS](/pricing#contact-sales)
:::
